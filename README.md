# SIBIR: Static Injection of Bibitems Into References

A streamlined, semi-automated workflow to convert modern `biblatex`/Biber citations into the hardcoded `\bibitem` format, which is easier to edit for JACoW (Joint Accelerator Conferences Website) publications. 

**SIBIR** handles the heavy lifting of extracting `biblatex` formatted bibliography entries imported from a `.bib` file and statically injecting them back into classic LaTeX bibliography.

## Prerequisites

To use this workflow, your working directory must contain the standard JACoW class and bibliography files, or have them available via TeX installation, alongside the extraction scripts.

Ensure you have the following in your folder:
* `jacow.cls`
* `jacow.cbx`
* `jacow.bbx`
* Your `.bib` database file
* Your main `.tex` draft

## Installation

Download the workflow files from this repository and place them into your working directory:
1.  `jacowbibextract.sty` (The LaTeX extraction package)
2.  `bib2items.cmd` (The main executable script)
3.  `Extract-BibItems.ps1` (PowerShell script for initial compilation)
4.  `Inject-BibItems.ps1` (PowerShell script for final code injection)

*Pro-Tip: For easy access on Windows, right-click `bib2items.cmd`, select "Create shortcut", and customize the shortcut's icon to make it look like a native application.*

## The Workflow

This pipeline uses your system's clipboard to bridge the gap between `biblatex` rendering and standard `\bibitem` injection.

1. **Launch the Pipeline:** Double-click `bib2items.cmd` (or your created shortcut). 
2. **First Compilation & Display:** The script will automatically trigger `Extract-BibItems.ps1`. This creates a temporary `.tex` file using `jacowbibextract.sty`, compiles it into a PDF containing your extracted references, and **automatically opens the PDF for you**.
3. **Copy References:** From the opened PDF, copy all the rendered `\bibitem` text together with `\begin{thebibliography}{99}` and `\end{thebibliography}` directly to your clipboard. (This ensures you capture the exact indentation and width calculations generated by the package).
4. **The Magic Step:** Return to the waiting command prompt window and **press any key**.
5. **Final Injection:** The `Inject-BibItems.ps1` script will automatically grab the `\bibitem`s from your clipboard, strip out all `bibtex`/`biber` keywords from your original `.tex` document, and inject the clipboard contents into a brand new `.tex` file (appended with `-final-bibitems`). 

The script stops here, leaving you with a perfectly prepped, JACoW-compliant LaTeX source file. It is up to you (the editor) to review the new file and run the final standard LaTeX compilation whenever you are ready!
